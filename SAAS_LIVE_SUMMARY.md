## 💡 Quick Start Customization

### 1. Change Brand Colors
Find and replace:
```tsx
from-blue-600 to-purple-600
↓
from-your-color to-your-color
```

### 2. Update Testimonials
Edit `TestimonialsSection.tsx`:
```typescript
const testimonials = [
  { quote: "Your testimonial", author: "Name", ... }
]
```

### 3. Modify Pricing
Edit `PricingSection.tsx`:
```typescript
const plans = [
  { name: "Your Plan", price: "$99", ... }
]
```

### 4. Connect a Form
Replace button onClick handlers with your form service.
